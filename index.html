🧠 What is Java?

Java is a high-level, object-oriented programming language used for building web, mobile, and desktop applications. It is platform-independent and widely used in enterprise software.

📦 Why Learn Java?

  • It's beginner-friendly and powerful.
  • Runs on all platforms (write once, run anywhere).
  • Used in Android apps, web development, and backend systems.
  • Huge community and support.

🔤 Basic Syntax

Here's a simple Java program:

public class HelloWorld {
  public static void main(String[] args) {
    System.out.println("Hello, World!");
  }
}
